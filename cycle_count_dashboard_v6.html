Methodology & grading scale
IRA % definition
IRA (Inventory Record Accuracy) measures the percentage of counted lines that fall within ABC tolerance thresholds:

A items: ±1% variance allowed
B items: ±3% variance allowed
C items: ±5% variance allowed

IRA % = Lines within tolerance ÷ Total lines counted
Grading scale
≥ 80% — Good · Within target
60–79% — Fair · Needs attention
< 60% — Poor · Action required

Completion: Green ≥85% · Amber ≥70% · Red <70%
IRA: Green ≥80% · Amber ≥60% · Red <60%
Target lines: Completion — 90%  ·  IRA — 80%
Accuracy methods compared
IRA % (shown) — Line-level pass/fail per ABC tolerance. Honest view; one missed line = one failure regardless of quantity.

Workbook accuracy — Σ Physical ÷ Σ System quantity. Quantity-weighted; large-volume items correct can mask many small misses. Shown for reference only.

IRA is the industry standard for cycle count program health.
